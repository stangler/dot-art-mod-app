

# 🎮 DotArt Mod



### Minecraft Java Edition 1.21.1 × NeoForge 21.1.172



画像ファイルを指定するだけで、マイクラの世界に

**高品質なブロックドットアートを自動生成・設置**できるModです。

**高度な色再現アルゴリズムと、大規模設置でも安定動作する分割パケット送信**を備えた実用ツールです。



---

## ✨ 主な特徴



- 🎨 **高品質ディザリング**: Floyd-Steinberg法により、限られたブロック数でも滑らかなグラデーションを再現。



- 👀 **リアルタイムプレビュー**: 設置前にGUI上でドット化の結果を確認可能。



- 🧱 **豊富なブロックパレット**: Wool、Concrete、Terracotta等から最適な色を自動選別。



- 📐 **柔軟なサイズ調整**: 16×16 から最大 64×64 までスライダーで直感的に変更。



- 🚀 **パケット分割送信**: 4000ブロックを超える設置でも、サーバー制限に抵触せず安定して配置。



- 🖥 **Dev Container対応**: Linuxパス形式での画像読み込みを完全サポート。



- 🧭 **設置方向モード**: 壁状（垂直）・床状（水平）の2種類の設置方向をGUIから切り替え可能。

```bash

./gradlew build

```

`build/libs/dotartmod-1.0.0.jar`

## 📦 インストール



1. NeoForge をインストール。



2. `dotartmod-1.0.0.jar` を `.minecraft/mods/` フォルダに配置。



3. Minecraft を起動。



---



## 🪄 使い方



### ① 杖（Wand）を入手



以下のコマンド、またはクリエイティブタブから入手してください。



```bash

/give @s dotartmod:dot_art_wand

```



### ② GUIを開く



杖を持って空中で **右クリック** すると設定画面が開きます。



### ③ 画像を読み込む



画像の絶対パスを入力し、**「読み込み & プレビュー」**ボタンを押します。



例（Dev Container環境）：`/home/mcdev/images/pikachu.png`



### ④ 設置方向モードを選択



GUI上部の **「Mode: WALL / FLOOR」** ボタンで設置方向を切り替えます。



- **WALL**: プレイヤーの向いている方向に対して垂直な壁状に設置

- **FLOOR**: プレイヤーの向いている方向に対して水平な床状に設置



### ⑤ プレビュー確認 & 設置



画面下に表示されるドット絵を確認し、問題なければ**「設置開始」**を押します。



プレイヤーの向いている方向（前方5ブロック先）を起点に、選択したモードに応じて自動で設置されます。

## 🎨 アルゴリズム仕様



### 1. 色変換（Color Mapping）



人間の視覚特性に基づいた重み付きRGB距離算出による最近傍探索を行い、最も近いマイクラブロックを選択します。



### 2. ディザリング（Dithering）



**Floyd-Steinberg 誤差拡散法**を採用。



単色ブロックでは表現しきれない中間色を、周囲のピクセルに誤差を分散させることで疑似的に再現し、写真のような質感を維持します。



---



## 🧱 設置仕様



- **配置位置**: プレイヤー前方 5ブロック地点を起点

- **方向**: GUIのモードに応じて切り替え

  - **WALL**: プレイヤーの視線方向に合わせた垂直（壁状）配置

  - **FLOOR**: プレイヤーの視線方向に合わせた水平（床状）配置

- **透明度**: 画像の透過（Alpha値）を判別し、背景を自動カット

- **安定性**: 200ブロックごとにパケットを分割送信し、サーバーキックを防止
