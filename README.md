# 🎮 DotArt Mod
### Minecraft Java Edition 1.21.1 × NeoForge 21.1.172

画像ファイルを選ぶだけで、マイクラの世界に**自動でブロックドットアートを作成**できるMod！
**複数ブロックを使った高精度なドットアート生成に対応。**

---

## ✨ 主な特徴

- 🎨 画像から自動ドットアート生成
- 🧱 **ブロックID対応で正確な色再現**
- 📐 サイズ変更（16〜64）
- 👀 プレビュー表示
- ⚡ ワンクリックで自動設置
- 🖥 Dev Container 環境対応（パス入力）

---

## 📋 必要なもの

| ツール | バージョン |
|--------|-----------|
| Java   | 21 以上   |
| Minecraft | 1.21.1 |
| NeoForge | 21.1.172 |

---

## 🚀 ビルド方法

```bash
./gradlew build
````

```text
build/libs/dotartmod-1.0.0.jar
```

---

## 📦 インストール

1. NeoForge をインストール
2. `dotartmod-1.0.0.jar` を `.minecraft/mods/` に配置
3. Minecraft を起動

---

## 🪄 使い方

### ① 杖を入手

```bash
/give @s dotartmod:dot_art_wand
```

---

### ② 右クリックでGUIを開く

---

### ③ 画像を読み込む

Dev Containerの場合👇

```
/home/mcdev/images/sample.png
```

---

### ④ 「設置」ボタン

👉 自動でドットアートが生成される！

---

## 🧱 設置仕様（重要）

* プレイヤー前方3マスから開始
* 中央揃えで展開
* 下 → 上 に積み上げ
* 各ピクセルごとに最適なブロックを配置

---

## 🎨 色変換アルゴリズム

```text
距離 = 2R² + 4G² + 3B²
```

👉 人間の視覚に最適化された重み付き距離

---

## 📏 サイズ一覧

| サイズ   | ブロック数 |
| ----- | ----- |
| 16×16 | 256   |
| 32×32 | 1024  |
| 64×64 | 4096  |

---

## 📁 ソース構成

```
com/dotartmod/
├── client/screen/     ← GUI・画像処理
├── network/           ← パケット通信
├── server/            ← ブロック設置
├── util/              ← カラーパレット
└── item/              ← 杖アイテム
```

---

## ⚙️ アーキテクチャ

```
[クライアント]
画像 → 色変換 → (BlockPos + BlockID) 送信

[サーバー]
受信 → ブロック取得 → setBlock()
```

👉 **完全サーバー主導で安全に設置**

---

## ⚠️ 注意点

* 大きいサイズ（64×64）は一瞬ラグあり
* Dev Containerではファイル選択ダイアログは使えません

---

## 🛠 トラブルシューティング

### ビルド失敗

```bash
./gradlew --stop
./gradlew clean build
```

---

### 画像が読み込めない

* パスが正しいか確認
* コンテナ内のパスを使用

---

### ブロックが置かれない

* 空中や設置不可ブロックを避ける
