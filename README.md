# 🎮 DotArt Mod
### Minecraft Java Edition 1.21.1 × NeoForge 21.1.172

画像ファイルを指定するだけで、マイクラの世界に
**高精度なブロックドットアートを自動生成・設置**できるModです。

**複数ブロックID対応・大規模設置でも安定動作する実用ツール**として設計されています。

---

## ✨ 主な特徴

- 🎨 画像から自動ドットアート生成
- 🧱 **ブロックID指定による正確な色再現**
- 📐 サイズ変更（16〜64）
- 👀 プレビュー表示
- ⚡ ワンクリック自動設置
- 🚀 **パケット分割送信による大規模設置の安定化**
- 🖥 Dev Container 環境対応（パス入力方式）

---

## 📋 必要なもの

| ツール | バージョン |
|--------|-----------|
| Java | 21 以上 |
| Minecraft | 1.21.1 |
| NeoForge | 21.1.172 |

---

## 🚀 ビルド方法

```bash
./gradlew build
```

生成物：

```text
build/libs/dotartmod-1.0.0.jar
```

---

## 📦 インストール

1. NeoForge をインストール
2. `dotartmod-1.0.0.jar` を `.minecraft/mods/` に配置
3. Minecraft を起動

---

## 🪄 使い方

### ① 杖を入手

```bash
/give @s dotartmod:dot_art_wand
```

---

### ② 右クリックでGUIを開く

---

### ③ 画像を読み込む

Dev Container 環境の場合：

```text
/home/mcdev/images/sample.png
```

---

### ④ 「設置」ボタンを押す

👉 自動でドットアートが生成・設置されます。

---

## 🧱 設置仕様（重要）

- プレイヤー前方 **3ブロック** から開始
- **中央揃え**
- **下 → 上** に積み上げ
- 各ピクセルごとに最適なブロックを配置
- **4096ブロック（64×64）でも安定動作**

---

## 🎨 色変換アルゴリズム

```text
距離 = 2R² + 4G² + 3B²
```

👉 人間の視覚特性を考慮した重み付き距離による最近傍探索

---

## 📏 サイズ一覧

| サイズ | ブロック数 |
|------|-----------|
| 16×16 | 256 |
| 32×32 | 1024 |
| 64×64 | 4096 |

---

## 📁 ソース構成

```
com/dotartmod/
├── client/screen/     ← GUI・画像処理
├── client/            ← クライアント制御
├── network/           ← パケット通信
├── server/            ← ブロック設置
├── util/              ← カラーパレット
└── item/              ← 杖アイテム
```

---

## ⚙️ アーキテクチャ

```
[クライアント]
画像 → 色変換 → (BlockPos + BlockID) 分割送信

[サーバー]
受信 → ブロック取得 → setBlock()
```

👉 **完全サーバー主導で安全に設置**

---

## ⚠️ 動作仕様について

- 大規模設置時は内部的にパケットを分割して送信します
- 設置結果は一瞬で反映されます（Minecraftの同期仕様によるもの）
- フリーズやクラッシュは発生しません
- サイズ変更後に再読み込みせず設置しても安全です

---

## 🛠 トラブルシューティング

### ビルド失敗

```bash
./gradlew --stop
./gradlew clean build
```

---

### 画像が読み込めない

- パスが正しいか確認
- コンテナ内のパスを使用しているか確認

---

### ブロックが置かれない

- 設置不可ブロックや空中を避ける
- 十分なスペースを確保する
